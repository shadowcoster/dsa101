# FAANG DSA Roadmap

Use this as a topic-first practice bank for FAANG-style coding interviews. Each folder has a `QUESTIONS.md` checklist.

## How to Use

1. Start with `00-start-here`.
2. Solve topic by topic for pattern building.
3. Revisit missed problems after 3 days, 7 days, and 21 days.
4. For every solved problem, write:
   - brute force idea
   - optimized idea
   - time and space complexity
   - mistake notes

## Priority Rule

- `[IMP]`: highest-leverage problems to master first for FAANG-style interviews.
- High priority: solve until you can explain it without looking.
- Medium priority: solve once, then review pattern.
- Hard priority: attempt seriously, then study editorial if stuck after 45 minutes.
