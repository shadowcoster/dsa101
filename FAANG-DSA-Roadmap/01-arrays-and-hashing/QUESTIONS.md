# Arrays and Hashing

## Core

- [ ] [IMP] Contains Duplicate
  - Problem statement: Given an integer array, determine whether any value appears at least twice.
- [ ] [IMP] Valid Anagram
  - Problem statement: Given two strings, determine whether one string is an anagram of the other.
- [ ] [IMP] Two Sum
  - Problem statement: Given an integer array and a target value, return the indices of two distinct elements whose sum equals the target.
- [ ] [IMP] Group Anagrams
  - Problem statement: Given a list of strings, group together words that are anagrams of each other.
- [ ] [IMP] Top K Frequent Elements
  - Problem statement: Given an integer array and k, return the k most frequent values.
- [ ] [IMP] Product of Array Except Self
  - Problem statement: Given an integer array, return an array where each position contains the product of all other elements without using division.
- [ ] [IMP] Valid Sudoku
  - Problem statement: Given a partially filled 9x9 Sudoku board, determine whether the current placement is valid.
- [ ] [IMP] Encode and Decode Strings
  - Problem statement: Design methods to encode a list of strings into one string and decode it back exactly.
- [ ] [IMP] Longest Consecutive Sequence
  - Problem statement: Given an unsorted integer array, find the length of the longest run of consecutive values.
- [ ] Majority Element
  - Problem statement: Given an array, return the element that appears more than half of the time.
- [ ] Sort Colors
  - Problem statement: Given an array containing only three color values, sort it in-place so equal colors are grouped in order.
- [ ] [IMP] Missing Number
  - Problem statement: Given numbers from 0 to n with one value missing, return the missing value.
- [ ] Find All Numbers Disappeared in an Array
  - Problem statement: Given the required inputs for Find All Numbers Disappeared in an Array, return the expected result efficiently while handling edge cases.
- [ ] [IMP] Single Number
  - Problem statement: Given an array where every value appears twice except one, return the value that appears once.
- [ ] Intersection of Two Arrays
  - Problem statement: Given two arrays, return the unique values that appear in both arrays.
- [ ] Intersection of Two Arrays II
  - Problem statement: Given two arrays, return their intersection including duplicate occurrences.
- [ ] First Unique Character in a String
  - Problem statement: Given a string, return the index of the first character that appears exactly once.
- [ ] Ransom Note
  - Problem statement: Given a note string and a magazine string, determine whether the note can be built from magazine characters.
- [ ] Isomorphic Strings
  - Problem statement: Given two strings, determine whether characters from one string can be consistently remapped to form the other.
- [ ] Word Pattern
  - Problem statement: Given a pattern and a sentence, determine whether words follow the same one-to-one mapping as the pattern characters.
