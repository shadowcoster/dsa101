# Start Here

## Warmup Checklist

- [ ] [IMP] Two Sum done


  - Problem statement: Given an integer array and a target value, return the indices of two distinct elements whose sum equals the target.
- [ ] [IMP] Valid Anagram done


  - Problem statement: Given two strings, determine whether one string is an anagram of the other.
- [ ] [IMP] Contains Duplicate done


  - Problem statement: Given an integer array, determine whether any value appears at least twice.
- [ ] [IMP] Valid Palindrome done


  - Problem statement: Given a string, determine whether it reads the same forward and backward after ignoring case and non-alphanumeric characters.
- [ ] [IMP] Binary Search Done


  - Problem statement: Given a sorted array and a target value, return the target index if it exists, otherwise return -1.
- [ ] [IMP] Merge Two Sorted Lists pending


  - Problem statement: Given two sorted linked lists, merge them into one sorted linked list.
- [ ] [IMP] Valid Parentheses done


  - Problem statement: Given a string containing brackets, determine whether every opening bracket is closed in the correct order.
- [ ] [IMP] Best Time to Buy and Sell Stock


  - Problem statement: Given daily stock prices, choose one buy day and one later sell day to maximize profit.
- [ ] [IMP] Maximum Subarray


  - Problem statement: Given an integer array, find the contiguous subarray with the largest possible sum.
- [ ] [IMP] Climbing Stairs


  - Problem statement: Given n stairs and the ability to climb one or two steps at a time, count how many distinct ways there are to reach the top.
- [ ] [IMP] Reverse Linked List


  - Problem statement: Given the head of a singly linked list, reverse the list and return the new head.
- [ ] [IMP] Invert Binary Tree


  - Problem statement: Given the root of a binary tree, swap every left and right child and return the inverted tree.
- [ ] Flood Fill


  - Problem statement: Given an image grid, a starting cell, and a new color, recolor the connected region matching the starting color.
- [ ] [IMP] Number of Islands


  - Problem statement: Given a grid of land and water cells, count how many connected land components exist.
- [ ] [IMP] House Robber


  - Problem statement: Given house values in a line, compute the maximum amount that can be robbed without robbing adjacent houses.
- [ ] [IMP] Coin Change


  - Problem statement: Given coin denominations and an amount, return the minimum number of coins needed to make that amount, or -1 if impossible.
- [ ] [IMP] Course Schedule


  - Problem statement: Given courses and prerequisite pairs, determine whether all courses can be completed.
- [ ] [IMP] Merge Intervals


  - Problem statement: Given a list of intervals, merge all overlapping intervals and return the condensed list.
- [ ] [IMP] Kth Largest Element in an Array
  - Problem statement: Given an unsorted array and k, return the kth largest element.
- [ ] [IMP] Product of Array Except Self


  - Problem statement: Given an integer array, return an array where each position contains the product of all other elements without using division.

## Interview Habits

- [ ] Restate the problem clearly.
- [ ] Ask about constraints and edge cases.
- [ ] Give brute force first if useful.
- [ ] Explain the optimized pattern.
- [ ] Code with small helper functions.
- [ ] Test empty, one-element, duplicate, negative, and large inputs.
- [ ] Say time and space complexity.
- [ ] Practice 30 to 45 minute timed sessions.
