# Sources and Notes

This roadmap is a curated practice structure, not a direct copy of any single list. It combines common FAANG interview patterns with widely used public prep lists and current topic guidance.

## Selection Logic

- Added `[IMP]` to the highest-leverage questions after checking current 2026 FAANG prep guidance.
- Prioritized high-frequency interview patterns: arrays, strings, hash maps, two pointers, sliding window, trees, graphs, dynamic programming, heaps, intervals, tries, and design.
- Added harder variants for FAANG depth rounds.
- Added SQL and database questions for roles where screening includes data/database rounds.
- Kept each topic folder independent so you can practice by pattern.

## Why These Tags Matter

- Current FAANG prep guidance repeatedly emphasizes pattern coverage over random volume.
- Arrays/strings, trees/graphs, and dynamic programming remain the biggest buckets.
- Binary search, heaps, intervals, stacks, tries, union-find, and design questions are high-yield supporting areas.
- For best results, solve `[IMP]` questions first, then use the untagged questions as reinforcement and variants.
